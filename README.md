elib-ace
========

Elib for using an ACE editor


__Note__: For full screen to work, add this CSS:

```
.fullScreen .fullScreen-editor{ 
	height: auto!important;
	width: auto!important;
	border: 0;
	margin: 0;
	position: fixed !important;
	top: 0;
	bottom: 0;
	left: 0;
	right: 0;
	z-index: 10000
}

.fullScreen {
	overflow: hidden
}
```
